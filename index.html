Meu site em HTML

Este é o meu primeiro site em foi criado com o objetivo de estudar e praticar HTML .

A ideia é aprender como funciona a estrutura de uma página, entender os principais elementos e treinar a criação de conteúdo para web. O site é simples, mas é uma forma de colocar em prática tudo o que estou aprendendo. Com o tempo, pretendo adicionar novos recursos e deixar o visual mais bonito conforme evoluo nos estudos.
