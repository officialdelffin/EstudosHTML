Estudos de Tags

que são tags:

Tags são comandos usados no HTML para definir a estrutura e o conteúdo de uma página web. Elas indicam ao navegador como exibir textos, imagens, links, títulos, parágrafos e outros elementos. Normalmente, uma tag tem uma abertura e um fechamento, envolvendo o conteúdo que será exibido.

Tipos de tags

Tag b - Deixa o texto em negrito, mas sem dar ênfase especial ao conteúdo.
Tag strong - Deixa o texto em negrito e indica que é importante (semântica).
Tag u - Sublinha o texto.
Tag i - Deixa o texto em itálico, usado para palavras estrangeiras ou termos especiais.
Tag small - Deixa o texto menor que o tamanho normal.
Tag del - Mostra o texto como riscado, indicando que foi removido ou substituído.
Tag mark - Destaca o texto com uma marcação amarela, como se estivesse grifado.
Tag sup - Coloca o texto em posição superior, como em potências.
Tag sub - Coloca o texto em posição inferior, como em fórmulas químicas.
Tag hr - Cria uma linha horizontal de separação entre conteúdos.
Tag br - Quebra a linha, fazendo o texto continuar na linha de baixo.

Estudos Link relativo (URL relativa)

Os links raltivos são um pouco diferentes dos links absolutos. Eles fazem a navegação dentro de pastas e arquivos que estão dentro do projeto

Para que tenha a naveganção entre essas pastas e arquivos tambem ultilizamos a tag a junto com o atributo href, mas no lugar de colocarmos a url do site colocamos o caminho da pasta

Exemplo :

nomeDaPasta/nomeDoArquivo

Para voltar não basta apenas colocar o nome da pasta anterior, ela funcionara assim somente se o arquivo estiver na mesma pasta , se tiver em pastas diferentes deve voltar a pasta ate chegar ao local desejado.

Exemplo :

../nomeDoArquivo

Estudos de tabelas

Em HTML, a tag table é usada para criar tabelas. Cada linha é definida com tr, os títulos das colunas com th e os dados com td. O th aparece em negrito e centralizado, enquanto o td exibe os dados normalmente.

Para fazer os espaços entre os elementos da tabela usamos o width e a porcetagem. E para colocar bordas na tabela usamos o border e um numero da grossura da borda.

Estudos de listas ordenadas e não ordenadas


Listas ordenadas

No HTML, lista ordenada é uma lista em que os itens aparecem em uma ordem numérica (1, 2, 3...). Usa-se a tag ol (ordered list).

Exemplo :

  1. Arroz
  2. feijão
  3. Frutas
  4. Carne

Listas não ordenadas

Já a lista desordenada mostra os itens com marcadores (bolinhas) sem seguir ordem. Usa-se a tag ul (unordered list). Exemplo :

  • Arroz
  • feijão
  • Frutas
  • Carne
