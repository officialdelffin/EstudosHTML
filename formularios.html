Nos formularios é possivel fazer a interação com o usuario e uma das funções do formulario é permitir que o usuario faça o input de algo no site como texto dentro outros, mas para isso deve se utilizar a tag input junto com o seu atributo type que define o tipo do input. No input tambem é possivel definir um hint, que indica ao usuário o que ele deve inserir naquele input, e para definir esse input ultilizamos o placehloder.

Para definir um nome para o input é necessario usar a tag label, onde você define o nome da caixa de entrada. Com o nome da caixa de entrada definida, também é possivél fazer um evento de clique, onde ao clicar no nome ele abre a caixa de entrada e deixa você escrever, mas para isso deve-se usar o atributo id no input junto com o for no label.

O que seria um cadatros sem um botão que envie o dados? Para criar um botao que envie os dados ultilizamos a tag input com o tipo submit para definir que é um buttom e com o atributo value para definir o nome do botao.
