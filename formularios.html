Voltar a pagina principal

Nos formularios é possivel fazer a interação com o usuario e uma das funções do formulario é permitir que o usuario faça o input de algo no site como texto dentro outros, mas para isso deve se utilizar a tag input junto com o seu atributo type que define o tipo do input. No input tambem é possivel definir um hint, que indica ao usuário o que ele deve inserir naquele input, e para definir esse input ultilizamos o placehloder.

Para definir um nome para o input é necessario usar a tag label, onde você define o nome da caixa de entrada. Com o nome da caixa de entrada definida, também é possivél fazer um evento de clique, onde ao clicar no nome ele abre a caixa de entrada e deixa você escrever, mas para isso deve-se usar o atributo id no input junto com o for no label.

O que seria um cadatros sem um botão que envie o dados? Para criar um botao que envie os dados ultilizamos a tag input com o tipo submit para definir que é um buttom e com o atributo value para definir o nome do botao.

Para obrigar que um usuario escreva todos os dados que precisam ser preenchidos usando dentro da tag input o atributo required, que não deixa os dados serem enviados caso ele não esteja preenchido.

Tipo radio em input

O input type="radio" em HTML cria botões circulares para seleção única, onde, ao usar o mesmo atributo name, apenas uma opção pode ser escolhida dentro do grupo, sendo ideal para perguntas com múltiplas alternativas, como sexo, forma de pagamento ou preferências.

Para usar o radio, o tipo do input tem que ser definido como radio, ele também tem que ter um identificador unico, um name e um value.

Para definir o nome na frente do radio deve se criar um label usando o atributo for com o id do input desejado.

Tipo checkbox em input

O input type="checkbox" em HTML cria caixinhas que o usuário pode marcar ou desmarcar livremente, permitindo várias seleções ao mesmo tempo; cada checkbox marcado envia seu value no envio do formulário, e todos podem compartilhar o mesmo name se representarem um mesmo grupo de dados. Select em formulario

O select cria um menu suspenso com várias option, permitindo ao usuário escolher uma (ou mais, se usar multiple); o valor selecionado é enviado no formulário com o nome definido pelo name. Cada option tem um value, que é o que será enviado, e um texto visível que aparece na lista.

Para fazer isso você deve usar o select para criar uma lista de opções, colocando várias option dentro dele; define um name no select para identificar o campo e um value em cada option para indicar o valor que será enviado quando a opção for escolhida.
